# Aventures de Robòtica

Un joc educatiu dissenyat especialment per a infants (a partir de 3-4 anys) per introduir-los en el pensament lògic i la programació de forma visual i intuïtiva.

## 🚀 Característiques

*   **2 Modes de Joc**:
    *   **Pilotatge**: Control directe del personatge per la graella fent servir botons a la pantalla o les fletxes del teclat.
    *   **Programació**: Crear seqüències de moviments (Amunt, Avall, Dreta, Esquerra) de forma anticipada per portar el personatge a l'objectiu, a l'estil Bee-Bot però en un pla absolut 2D (més fàcil d'entendre per als més petits que els girs relatius).
*   **4 Personatges i Entorns**:
    *   🐢 **Tortuga:** Navega per l'aigua esquivant murs de maons per arribar a l'enciam.
    *   💃 **Ballerina:** Balla per sobre el parquet cap a la nota musical.
    *   🛵 **Moto Kawaii:** Condueix per la carretera esquivant senyals d'obres cap a la bateria.
    *   🚀 **Nau Espacial:** Vola per l'espai fosc esquivant asteroides cap a l'estrella.
*   **Nivells de Dificultat**:
    *   **Aventura (Ordre Normal)**: 18 nivells progressius. Comença en graelles petites (4x4) sense obstacles i va augmentant la mida (fins a 8x8) afegint petits laberints.
    *   **Nivell Aleatori**: Tria un nivell qualsevol a l'atzar per fer partides infinites!
*   **Disseny Kids-Friendly**: Interfície gran, acolorida, basada en emojis i "glassmorphism", pensada per ser utilitzada fàcilment en tauletes.

## 💻 Instal·lació Local (Per a desenvolupadors)

Si vols descarregar el codi i modificar-lo per afegir els teus propis nivells o personatges:

1.  Clona aquest repositori:
    ```bash
    git clone https://github.com/guillemcasamayor/robotica_aeris.git
    ```
2.  Obre l'arxiu `index.html` amb qualsevol navegador web. No cal cap servidor ni procés de construcció, és pur HTML, CSS i Javascript!

## 🛠️ Tecnologies
*   HTML5
*   CSS3 (Variables CSS, Flexbox, CSS Grid)
*   Vanilla Javascript (Sense frameworks externs)
